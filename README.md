# IoT Data Simulator 🚀

![License](https://img.shields.io/badge/License-MIT-blue?style=for-the-badge)
![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)

Một công cụ giả lập dữ liệu IoT nhẹ nhàng, giúp sinh ra các dữ liệu cảm biến ngẫu nhiên (nhiệt độ, độ ẩm, nhịp tim) để phục vụ cho việc kiểm thử các hệ thống IoT Dashboard.

## Tính năng
- Sinh dữ liệu ngẫu nhiên thời gian thực.
- Hỗ trợ xuất dữ liệu ra Console.
- Sẵn sàng tích hợp với MQTT hoặc HTTP REST API (trong tương lai).

## Cài đặt
```bash
python main.py
```
